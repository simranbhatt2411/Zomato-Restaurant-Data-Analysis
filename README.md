# Delhi Restaurants Data Analysis

## Project Overview
This project focuses on analyzing restaurant data for Delhi using Python-based data analysis and visualization techniques. The objective of the project is to explore restaurant distribution, identify high-density restaurant areas, analyze customer ratings and pricing trends, and generate meaningful business insights from the dataset.

The analysis was performed using Jupyter Notebook with libraries such as Pandas, Matplotlib, and Seaborn.

---

## Objectives
- Analyze restaurant data for Delhi
- Identify areas with high restaurant density
- Explore dining and delivery rating distributions
- Analyze pricing patterns across localities
- Identify popular restaurant categories
- Generate business insights using data visualization

---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

---

## Dataset Information
The dataset contains restaurant-related information including:
- Restaurant Name
- Locality
- Dining Rating
- Delivery Rating
- Pricing for Two
- Restaurant Category

The dataset was collected from a publicly available restaurant data source.

---

## Data Cleaning Process
The following preprocessing steps were performed:
- Standardized column names
- Removed missing values from important columns
- Checked dataset structure and data types
- Cleaned inconsistent records for accurate analysis

---

## Analysis Performed

### 1. Restaurant Density Analysis
- Identified top localities with the highest number of restaurants
- Visualized restaurant concentration using bar charts

### 2. Rating Analysis
- Analyzed dining rating distribution
- Explored customer satisfaction trends

### 3. Pricing Analysis
- Studied pricing distribution across restaurants
- Identified premium and budget-friendly restaurant areas

### 4. Cuisine / Category Analysis
- Identified most common restaurant categories
- Explored customer dining preferences

### 5. Correlation Analysis
- Analyzed relationships between pricing and ratings
- Generated heatmaps and scatter plots for visualization

---

## Key Insights
- Commercial and central areas have the highest restaurant density.
- Most restaurants maintain ratings between 3.5 and 4.5.
- Delhi contains a mix of premium and affordable dining options.
- Popular food hubs indicate strong customer demand and market competition.
- Pricing and ratings show moderate correlation in customer dining behavior.
